✅ Exercício 5 – Substituir um item por outro em todo o array
💻 Enunciado:
Dado este array:
let frutas = ['maçã', 'banana', 'laranja', 'banana', 'uva', 'banana'];
Substitua todas as 'banana' por 'morango', usando:
indexOf()
splice()
🎯 Resultado esperado:
['maçã', 'morango', 'laranja', 'morango', 'uva', 'morango']
✅ Dica:
Você vai usar um while para repetir enquanto indexOf('banana') retornar algo diferente de -1. Dentro do laço, use splice() para trocar o valor no índice encontrado.
